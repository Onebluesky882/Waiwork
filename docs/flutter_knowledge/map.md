ลักษณะสำคัญของ Map:

เข้าถึงข้อมูลผ่าน key ไม่ใช่ index

สามารถใช้ key เป็น String, int หรือ Object อื่น ๆ

Map<String, int> ages = {
'Alice': 25,
'Bob': 30,
'Charlie': 28
};

print(ages['Bob']); // 30

List (Array)

List<String> fruits = ['Apple', 'Banana', 'Cherry'];

print(fruits[1]); // Banana
